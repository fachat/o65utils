o65-dump
========

This repository contains the `o65-dump` tool for dumping the contents
of 6502 relocatable binaries in the [o65 format](http://www.6502.org/users/andre/o65/).

Building
--------

```
mkdir build
cd build
cmake ..
make
```

Using
-----

```
o65-dump hello.o65
```
